# 一键下载全部书籍功能

## 功能描述
在微信读书下载器的主页添加了一键下载全部书籍的功能，用户可以方便地下载书架上的所有书籍。

## 新增功能
1. **一键下载按钮**：在主页标题栏添加了"一键下载全部"按钮
2. **下载进度显示**：显示下载进度百分比和已下载/总书籍数量
3. **防频繁请求**：每本书之间延迟1秒，避免请求过于频繁导致的问题
4. **确认对话框**：在开始下载前会提示用户确认操作

## 技术实现
1. 添加了新的响应式变量：
   - `isDownloadingAll`：标识是否正在下载全部书籍
   - `downloadProgress`：下载进度百分比
   - `downloadedCount`：已下载的书籍数量
   - `totalBooks`：总书籍数量

2. 实现了`downloadAllBooks`异步函数：
   - 检查书架是否有书籍
   - 通过确认对话框获得用户确认
   - 按顺序逐本下载书籍（每本下载完成后间隔1秒）
   - 实时更新下载进度
   - 下载完成后显示成功消息

## 用户体验优化
- 使用了加载状态和进度条，让用户了解下载进度
- 避免了同时发起多个请求导致的问题
- 添加了确认步骤，防止误操作

## 使用方法
1. 登录应用并加载书架
2. 点击"一键下载全部"按钮
3. 确认下载操作
4. 等待下载完成，查看进度条和统计信息

这个功能大大提高了批量下载书籍的效率，用户无需逐本点击下载。
